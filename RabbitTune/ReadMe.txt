このソフトウェアは、様々なフォーマットの音楽ファイルを、簡単な操作で再生、および管理
するために作成されたソフトウェアです。フリーソフトのため、無料で使用できます。

●動作環境
　・Microsoft Windows 8.1/10/11（日本語環境）
　・Microsoft .NET Framework 4.8 以降
